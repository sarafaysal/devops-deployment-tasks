# Java Deployment using Apache Tomcat

## Objective

Deploy a Java application server using Apache Tomcat.

## Technologies Used

- Ubuntu 24.04 (WSL2)
- Java JDK
- Apache Tomcat 10

## Steps Performed

1. Installed Java JDK.
2. Installed Apache Tomcat.
3. Started the Tomcat service.
4. Verified Tomcat is running.

## Testing

- Java Version:
  - `java -version`

- Tomcat Status:
  - `sudo service tomcat10 status`

- Browser:
  - `http://localhost:8080`

## Result

Tomcat server was installed and is running successfully.
